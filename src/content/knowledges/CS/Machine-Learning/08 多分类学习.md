## 多分类学习介绍
标签空间变成：$\mathcal{Y}=\{1,2,\dots,K\},\quad K\ge 3$
我们要建模的是一个条件分布向量：$$\boldsymbol{\eta}(x)=\big(P(y=1\mid x), P(y=2\mid x), \dots, P(y=K\mid x)\big)$$满足：$P(y=k\mid x)\ge 0,\qquad \sum_{k=1}^K P(y=k\mid x)=1$
对应的 Bayes 最优决策是：$$\hat y^*(x)=\arg\max_{k\in\{1,\dots,K\}} P(y=k\mid x)$$
## Softmax
对每个类别 $k \in \{1, \dots, K\}$，定义一个线性分数：$s_k(x) = w_k^\top x + b_k$
把所有类别分数堆成向量：$s(x) = \begin{bmatrix} s_1(x) \\ s_2(x) \\ \vdots \\ s_K(x) \end{bmatrix}$
Softmax 把分数向量变成概率向量：$$P(y=k \mid x) = \frac{e^{s_k(x)}}{\sum_{j=1}^K e^{s_j(x)}}$$$\sum_{j=1}^K e^{s_j(x)}$ 是归一化常数, 作用是把所有未归一化强度压缩成和为 1 的概率 ：$\sum_{k=1}^K P(y=k \mid x) = 1$
$$\hat{y}(x) = \arg\max_k s_k(x)$$

两个类别的概率比值（Odds Ratio）满足：$$\frac{P(y=i \mid x)}{P(y=j \mid x)} = \frac{e^{s_i(x)}}{e^{s_j(x)}} = e^{s_i(x) - s_j(x)}$$任意两类 $i, j$ 的 **Log-odds** 就是：
$$\log \frac{P(y=i \mid x)}{P(y=j \mid x)} = (w_i - w_j)^\top x + (b_i - b_j)$$Softmax Regression 实际上在建模两两对数赔率的线性结构

同时几何上我们可以发现 类别 $i$ 与 $j$ 的分界面时 $w_i^\top x + b_i = w_j^\top x + b_j$
这是一条超平面 $$(w_i - w_j)^\top x + (b_i - b_j) = 0$$ 因此，Softmax Regression 仍然是**线性分类器**，只是它不是用一个超平面切分两类，而是由多个线性分数共同划分空间
每个类别的决策区域是若干线性不等式交出来的凸区域：
$$\{x \mid s_k(x) \ge s_j(x), \ \forall j \neq k\}$$
从几何上看，多分类线性模型是在输入空间中形成一个分段线性 (**piecewise-linear**) 的 **Voronoi-like**划分


## multinomial likelihood 与 multiclass cross-entropy
### 似然函数分析
1.单样本的类别分布

1)对样本 $x_i$，模型的输出类别概率：
$$\pi_k(x_i; \theta) = P(y_i = k \mid x_i; \theta), \quad k=1, \dots, K$$
并满足归一化条件：
$$\sum_{k=1}^K \pi_k(x_i; \theta) = 1$$
2)label的分布
用 **one-hot** 向量表示标签，定义 $y_{ik}$：
$$y_{ik} = \begin{cases} 1, & \text{若样本 } i \text{ 的真实类别是 } k \\ 0, & \text{否则} \end{cases}$$
则对每个样本，都有：
$$\sum_{k=1}^K y_{ik} = 1$$
即向量中只有一个位置为 $1$

2.单样本 Likelihood
$$P(y_i \mid x_i; \theta) = \prod_{k=1}^K \pi_k(x_i; \theta)^{y_{ik}}$$
假设样本 $i$ 的真实类别是 $c$，则 $y_{ic}=1$，而对于所有 $k \neq c$，$y_{ik}=0$。上式展开为：

$$\prod_{k=1}^K \pi_k^{y_{ik}} = \pi_c^1 \cdot \prod_{k \neq c} \pi_k^0 = \pi_c \cdot 1 = \pi_c$$

3.整个数据集的 Likelihood
假设样本间条件独立，整个训练集的似然为：

$$L(\theta) = \prod_{i=1}^n P(y_i \mid x_i; \theta) = \prod_{i=1}^n \prod_{k=1}^K \pi_k(x_i; \theta)^{y_{ik}}$$

取对数似然 (Log-Likelihood)：

$$\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \sum_{k=1}^K y_{ik} \log \pi_k(x_i; \theta)$$

最大似然估计（MLE）即：

$$\hat{\theta} = \arg\max_\theta \ell(\theta)$$

最小化**负对数似然 (NLL)**，定义损失函数：

$$\mathcal{L}(\theta) = - \sum_{i=1}^n \sum_{k=1}^K y_{ik} \log \pi_k(x_i; \theta)$$

这就是 **Multiclass Cross-Entropy (多分类交叉熵)** 的经验风险形式
我们可以观察对与单个样本来说 由于 one-hot 标签的特性，只有真实类 $c$ 的项保留：

$$\mathcal{L}_i = - \log \pi_c(x_i; \theta)$$
**只惩罚真实类别的预测概率不够大**
- 如果真实类概率 $\pi_c$ 接近 $1$，损失趋于 $0$
- 如果真实类概率 $\pi_c$ 接近 $0$，损失趋于 $+\infty$

### 损失函数具体形式
Softmax 定义代入交叉熵损失
对于样本 $i$，若其 Logit 向量为 $s_i$，则第 $k$ 类的预测概率为：

$$\pi_k(x_i; \theta) = \frac{e^{s_{ik}}}{\sum_{j=1}^K e^{s_{ij}}}$$
对与单个样本来说 $$\mathcal{L}_i = -\sum_{k=1}^K y_{ik}s_{ik} + \log \sum_{j=1}^K e^{s_{ij}}$$

若样本 $i$ 的真实类别是 $c$（即 $y_{ic}=1$），损失进一步简化为：

$$\mathcal{L}_i = -s_{ic} + \log \sum_{j=1}^K e^{s_{ij}}$$
我们可以看见损失函数的两个部分:
- **$-s_{ic}$（推动项）**：直接推动真实类分数 $s_{ic}$ 变大
- **$\log \sum e^{s_{ij}}$（竞争调节项）**：这是一个 **Log-Sum-Exp** 项 它起到了归一化作用，防止模型通过无脑抬高所有类别的分数来降低损失 它强制要求真实类分数必须**相对**于其他类更高

### 梯度分析
对单样本损失 $\mathcal{L}$ 关于第 $k$ 个 Logit $s_k$ 求导，：

$$\frac{\partial \mathcal{L}}{\partial s_k} = \pi_k - y_k$$
- **对于真实类 $c$ ($y_c = 1$)**：
    $$\frac{\partial \mathcal{L}}{\partial s_c} = \pi_c - 1$$
只要预测概率 $\pi_c < 1$，梯度就为负 在梯度下降中，分数 $s_c$ 会被推向更大
    
- **对于非真实类 $k \neq c$ ($y_k = 0$)**：
    
    $$\frac{\partial \mathcal{L}}{\partial s_k} = \pi_k$$
由于 $\pi_k > 0$，梯度为正 在梯度下降中，这些错误类别的分数 $s_k$ 会被抑制变小 

**模型更新的方向等于“预测分布”减去“真实分布**”

输出分布不断向真实标签的脉冲分布靠近
