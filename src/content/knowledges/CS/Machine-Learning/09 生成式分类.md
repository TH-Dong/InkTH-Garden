## 判别式模型 vs 生成式模型
+ 判别式模型:Discriminative Model
直接建模 $P(\text{class}|x)$  或者说直接学习决策函数 $f(x)\rightarrow y$ 
eg.logistic Regression,SVM,neural network.
+ 生成式模型:Generative Model
我先去学会每一个class怎么去生成相应的data(feature),然后再利用这个进行分类
它需要去建模$P(x|y)$ (==类条件概率==**每个类别y的数据长什么样,出现feature x的概率是多少**) 和 $P(y)$ (==先验概率==**每个类别出现的概率是多少**) 然后根据Bayes Theorem:$$P(y|x)=\frac{P(x|y)P(y)}{P(x)}$$e g.Naive Bayes, Gaussian Discriminant Analysis, Hidden Markov Model, Gaussian Mixture Model（在更广义生成建模里） 
Eg.**判别式可以理解为: 学习一条边界,把两类物体进行分开**
**生成式理解为: 学习两类的各自形状是什么样子的,然后看x更像是哪一个类别**
❌ 误区：生成式模型只是“绕远路”  
✔️ 实际上它提供了更丰富的信息（数据分布）

## 贝叶斯决策理论（Bayesian Decision Theory）
当我们得到$P(y|x)$后 随之而来的问题是 如何去做decision?
我们不能单纯的看预测的$y$对不对就行了
因为你对的预测错误和错误的预测正确在现实生活中代价往往是不一样的
例如患病者检查为不患病显然是要更加的严重的

1.设：输入特征：$x \in \mathcal{X}$ , 真实类别：$y \in \mathcal{Y}$ , 我们采取的决策或动作：$a \in \mathcal{A}$
在普通分类里，通常动作空间和类别空间相同，即：
$$\mathcal{A} = \mathcal{Y}$$

给定一个输入 $x$，因为真实 $y$ 还不确定，所以动作 $a$ 的好坏要看**条件期望损失**：
$$R(a \mid x) = \mathbb{E}[L(a, y) \mid x]$$
这叫做**条件风险 (Conditional Risk)**
把期望展开：
$$R(a \mid x) = \sum_{y \in \mathcal{Y}} L(a, y) P(y \mid x)$$
最优动作定义为使条件风险最小的动作：
$$a^*(x) = \arg\min_{a \in \mathcal{A}} R(a \mid x)$$

2.**0-1 损失下会退化成“选后验概率最大的类”**
在 0-1 损失下：

$$L(k, y) = \begin{cases} 0, & k = y \\ 1, & k \neq y \end{cases}$$

那么条件风险为：

$$R(k \mid x) = \sum_y L(k, y) P(y \mid x)$$

因为只有在 $y \neq k$ 时损失为 $1$，所以：

$$R(k \mid x) = \sum_{y \neq k} P(y \mid x)$$

而所有类别后验概率和为 $1$，因此：

$$R(k \mid x) = 1 - P(y=k \mid x)$$

于是**最小化风险等价于最大化后验概率**：

$$a^*(x) = \arg\min_k \big(1 - P(y=k \mid x)\big) = \arg\max_k P(y=k \mid x)$$

3.只要你能建模 **类条件分布 $P(x \mid y)$** 和 **先验分布 $P(y)$**，就足以完成最优分类决策
这就是 MAP 分类规则（Maximum A Posteriori）：

$$\hat{y} = \arg\max_y P(y \mid x)$$

做 MAP 分类时：

$$\hat{y} = \arg\max_y P(y \mid x) = \arg\max_y \frac{P(x \mid y)P(y)}{P(x)}$$

因为对于固定的 $x$，$P(x)$ 与类别 $y$ 无关，所以可去掉分母：

$$\hat{y} = \arg\max_y P(x \mid y)P(y)$$
4.**二分类下的阈值形式：不一定是 0.5!**

 在对称 0-1 损失下
	设二分类 $y \in \{0, 1\}$，使用 0-1 损失：
	- 预测 $1$ 的风险：$R(1 \mid x) = P(y=0 \mid x)$
	- 预测 $0$ 的风险：$R(0 \mid x) = P(y=1 \mid x)$
	    当 $P(y=1 \mid x) > 0.5$ 时预测 $1$
	    

在非对称损失下
	设损失矩阵为：$L(1, 0) = \lambda_{10}$（误报代价），$L(0, 1) = \lambda_{01}$（漏报代价）
	- 预测 $1$ 的风险：$R(1 \mid x) = \lambda_{10} P(y=0 \mid x)$
	- 预测 $0$ 的风险：$R(0 \mid x) = \lambda_{01} P(y=1 \mid x)$
	选择 $1$ 的条件是 $R(1 \mid x) < R(0 \mid x)$，代入 $P(y=0 \mid x) = 1 - P(y=1 \mid x)$ 整理得：
	
	$$P(y=1 \mid x) > \frac{\lambda_{10}}{\lambda_{01} + \lambda_{10}}$$

**结论**：分类阈值不是天然的 $0.5$，而是由错误代价的比例决定的 在医疗或安防等漏报代价极大的场景中，阈值通常会远低于 $0.5$

## 朴素贝叶斯 (Naive Bayes)
### 核心任务
先建模类条件分布$P(x|y)$, 再乘上先验
但是实际上这个任务很艰难, 高维的联合分布往往很难进行建模
Eg. 对于$d$维二值feature 的$x_i\in \{0,1\}$ 一共有$2^d$种可能情况,这种指数增长 以至我们不可能为每一种feature建模一种概率

### 朴素贝叶斯的数学建模:
设类别空间为$y \in \{1, 2, \dots, K\}$, 输入为$x = (x_1, \dots, x_d)$
Naive Bayes 的两个核心建模对象：
**先验分布**
$$P(y=k) = \pi_k, \quad k=1, \dots, K$$
满足 $\pi_k \ge 0, \qquad \sum_{k=1}^K \pi_k = 1$
**类条件分布**
假设在给定类别 $y=k$ 时，各特征条件独立：
$$P(x \mid y=k) = \prod_{j=1}^d P(x_j \mid y=k)$$

于是联合分布写成：
$$P(x, y=k) = P(y=k)P(x \mid y=k) = \pi_k \prod_{j=1}^d P(x_j \mid y=k)$$
后验概率为：
$$P(y=k \mid x) = \frac{\pi_k \prod_{j=1}^d P(x_j \mid y=k)}{\sum_{\ell=1}^K \pi_\ell \prod_{j=1}^d P(x_j \mid y=\ell)}$$

最终分类规则： $\hat{y} = \arg\max_{k} P(y=k \mid x)$
等价于$$\hat{y} = \arg\max_{k} \pi_k \prod_{j=1}^d P(x_j \mid y=k)$$
实际计算时通常取对数：
$$\hat{y} = \arg\max_{k} \left[ \log \pi_k + \sum_{j=1}^d \log P(x_j \mid y=k) \right]$$
可以分析这个最值项的组合
- 先验项 $\log \pi_k$：类别本身的基础倾向
- 每个特征贡献一个局部证据 $\log P(x_j \mid y=k)$

这也是为什么**朴素贝叶斯在文本分类中特别自然**：
一篇文档是否属于某一类，可以理解为“每个词都给这个类别投一票”，最后把所有词的对数证据加起来

用一个很粗暴的结构假设,换来了可估计性 一旦知道类别，特征之间就再也没有相互作用了
(假设二值feature) 原参数规模$O(2^d)$ 现在$O(d\times|\mathcal Y|)$ 指数变线性 
+ 假设很强, 所以bias很大
+ ==参数很少, 方差小, 所以在小样本的时候很稳定==

### 朴素贝叶斯的常见模型与参数估计
我们有不同的办法去建模 类条件分布 $P(x_j|y)$

1)Bernoulli Naive Bayes
分析**二值特征** (❕特征是二值,不是二分类) 某个词是否出现,某个属性是否出现
$$P(x \mid y=k) = \prod_{j=1}^d \theta_{jk}^{x_j} (1 - \theta_{jk})^{1-x_j}$$
2)Multinomial Naive Bayes
适合计数型特征 文本分类
假设在类别k 不同词的生成概率为 $\phi_{1k}, \phi_{2k}, \dots, \phi_{dk}$ 满足 $\sum_{j=1}^d \phi_{jk} = 1$
$$P(x \mid y=k) \propto \prod_{j=1}^d \phi_{jk}^{x_j}$$
3)Guassian Naive Bayes
适合特征分布是continuous的
给定类别 $y=k$ 下，每个特征独立且服从一维高斯分布：$x_j \mid y=k \sim \mathcal{N}(\mu_{jk}, \sigma_{jk}^2)$
$$P(x_j \mid y=k) = \frac{1}{\sqrt{2\pi\sigma_{jk}^2}} \exp\left( -\frac{(x_j - \mu_{jk})^2}{2\sigma_{jk}^2} \right)$$**每个特征都有自己在每个类别下的均值和方差**

**Naive Bayes的参数估计**

==Naive Bayes的参数估计往往很简单 甚至有闭式解 不用进行复杂的迭代优化==
给定训练集$\{(x^{(i)}, y^{(i)})\}_{i=1}^n$ , 类别 $k$ 的样本数记作 $n_k$，MLE能够得到最佳估计
$$\hat{\pi}_k = \frac{n_k}{n}$$

1)Bernoulli Naive Bayes 的参数估计
MLE可以得到$$\hat{\theta}_{jk} = \frac{\sum_{i:y^{(i)}=k} x_j^{(i)}}{n_k}$$在所有类别为 $k$ 的样本中，第 $j$ 个特征出现的比例
2)Multinomial Naive Bayes 的参数估计
$x_j^{(i)}$ 是第 $i$ 个样本中第 $j$ 个词的计数 那么类别 $k$ 下第 $j$ 个词的概率可估计为：
$$\hat{\phi}_{jk} = \frac{\sum_{i:y^{(i)}=k} x_j^{(i)}}{\sum_{i:y^{(i)}=k}\sum_{t=1}^d x_t^{(i)}}$$
**分母**：类别 $k$ 的所有文档中，所有词一共出现了多少次
3)Gaussian Naive Bayes 的参数估计
对类别 $k$、特征 $j$，估计均值和方差：

$$\hat{\mu}_{jk} = \frac{1}{n_k} \sum_{i:y^{(i)}=k} x_j^{(i)}$$

$$\hat{\sigma}_{jk}^2 = \frac{1}{n_k} \sum_{i:y^{(i)}=k} (x_j^{(i)} - \hat{\mu}_{jk})^2$$

也是按类、按特征分别做频率统计或矩估计
### 零概率和平滑
如果对于某个类别 $k$ ,某一个feature $j$ 从来没有出现过 
那么按照MLE $\hat{\phi}_{jk}=0$
那么对于某个新的样本 一旦这个词出现 $$P(x \mid y=k) \propto \phi_{jk}^{x_j} = 0$$一个没见过的词，就能把整个类别的概率直接乘成 0 这个显然过于极端
我们常见的是做Laplace Smoothing 以Multinomial Naive Bayes为例
加一平滑: $$\hat{\phi}_{jk} = \frac{N_{jk} + 1}{\sum_{t=1}^d N_{tk} + d}$$
给每个词加1(伪计数) 分母加d保证归一化
加 $\alpha$ 平滑:$$\hat{\phi}_{jk} = \frac{N_{jk} + \alpha}{\sum_{t=1}^d N_{tk} + \alpha d}$$
==平滑本质上就是在数据之外 再引入一个温和的prior 避免最大似然把未观测事件的概率压成 0==

Naive Bayes的适用场景
+ 高维、稀疏、小样本
	文本分类、垃圾邮件识别、主题分类
	特征维度高, 每个样本只激活少量特征, 简单频率统计往往就很有效
+ 需要快速训练和快速基线
+  生成机制具有可分解结构时

局限
+ 条件独立假设过强
+ 概率校准通常不好
+ 连续特征的高斯假设可能不合理
## 高斯判别分析 (GDA)
GDA的建模假设:
每个class的数据点都在feature space里面形成一团
他们应该是在绕着某个点波动( 中心$\mu_k$ ) 但波动的方向和尺度或许有不同 ( 扩散结构$\Sigma_k$ ) 
于是分类的问题: 这个样本 $x$ 更像是从哪个类别对应的高斯分布里生成出来的？
### GDA数学模型
设类别$y \in \{1, 2, \dots, K\}$ , 输入 $x \in \mathbb{R}^d$
**先验分布**
$$P(y=k) = \pi_k, \qquad \sum_{k=1}^K \pi_k = 1$$
$\pi_k$ 表示类别 $k$ 的先验概率

**类条件分布**
GDA 假设：
$$x \mid y=k \sim \mathcal{N}(\mu_k, \Sigma_k)$$
密度函数为：
$$p(x \mid y=k) = \frac{1}{(2\pi)^{d/2} |\Sigma_k|^{1/2}} \exp\left( -\frac{1}{2}(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k) \right)$$

- $d$：特征维数
- $\mu_k \in \mathbb{R}^d$：类别 $k$ 的均值向量
- $\Sigma_k \in \mathbb{R}^{d \times d}$：类别 $k$ 的协方差矩阵
- $|\Sigma_k|$：协方差矩阵的行列式，反映总体体积尺度
- $\Sigma_k^{-1}$：精度矩阵，控制不同方向上的惩罚强度
+ 二次型 $(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k)$ : $x$ 到均值 $\mu_k$ 的 **Mahalanobis 距离平方**
>这和欧氏距离不同, 欧氏距离默认所有方向同等重要，但 Mahalanobis 距离会考虑：**各方向方差大小和 特征间相关性**
    
**后验概率**
由 Bayes 公式：$P(y=k \mid x) = \frac{\pi_k p(x \mid y=k)}{\sum_{\ell=1}^K \pi_\ell p(x \mid y=\ell)}$
定义判别函数：
$$\delta_k(x) = \log \pi_k + \log p(x \mid y=k)$$
代入高斯密度得：
$$\delta_k(x) = \log \pi_k - \frac{1}{2}\log |\Sigma_k| - \frac{1}{2}(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k) - \frac{d}{2}\log(2\pi)$$

最后一项 $-\frac{d}{2}\log(2\pi)$ 对所有类别都相同，分类时可忽略
$$\boxed{\delta_k(x) = \log \pi_k - \frac{1}{2}\log |\Sigma_k| - \frac{1}{2}(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k)}$$
判别函数的理解:
$$\delta_k(x) = \underbrace{\log \pi_k}_{\text{先验偏置}} - \frac{1}{2} \underbrace{\log |\Sigma_k|}_{\text{体积惩罚}} - \frac{1}{2} \underbrace{(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k)}_{\text{结构化距离惩罚}}$$

+ 体积惩罚: 分布散的点 每个点的密度就低 反之则高 因此可以作为扩散程度的修正
+ 距离惩罚: 样本 $x$ 离类别 $k$ 的中心 $\mu_k$ 有多远 按该类本身的波动结构校正后的远

### LDA
Linear Discriminant Analysis
LDA 的假设是：
$$x \mid y=k \sim \mathcal{N}(\mu_k, \Sigma)$$
即 不同类别的均值不相同 但是方差相同 **共享同一个协方差矩阵** $\Sigma$ 
我们通过计算:$$\delta_k(x) = \log \pi_k - \frac{1}{2}\log |\Sigma| - \frac{1}{2}(x - \mu_k)^\top \Sigma^{-1} (x - \mu_k)$$
因为 $\Sigma$ 对所有类别相同，$-\frac{1}{2}\log|\Sigma|$ 是公共项，可忽略 展开二次型：

$$(x - \mu_k)^\top \Sigma^{-1} (x - \mu_k) = x^\top \Sigma^{-1} x - 2\mu_k^\top \Sigma^{-1} x + \mu_k^\top \Sigma^{-1} \mu_k$$

代回得：

$$\delta_k(x) = \log \pi_k - \frac{1}{2}x^\top \Sigma^{-1} x + \mu_k^\top \Sigma^{-1} x - \frac{1}{2}\mu_k^\top \Sigma^{-1} \mu_k$$

其中 $-\frac{1}{2}x^\top \Sigma^{-1} x$ 对所有类别都相同，也可忽略 

$$\delta_k(x) = \mu_k^\top \Sigma^{-1} x - \frac{1}{2}\mu_k^\top \Sigma^{-1} \mu_k + \log \pi_k$$

这是关于 $x$ 的**线性函数** 所以 LDA 的决策规则是：

$$\hat{y} = \arg\max_k \left( \mu_k^\top \Sigma^{-1} x - \frac{1}{2}\mu_k^\top \Sigma^{-1} \mu_k + \log \pi_k \right)$$
类别 $a$ 和 $b$ 的分界面满足 $\delta_a(x) = \delta_b(x)$ 两边都是 $x$ 的Linear function，相减后就是超平面形式：
$$w^\top x + b = 0$$

**在多元高斯生成模型下，一旦所有类别共享协方差矩阵，判别函数里的二次项会完全抵消，只剩线性项**

### QDA
Quadratic Discriminant Analysis
QDA 的假设:
$$x \mid y=k \sim \mathcal{N}(\mu_k, \Sigma_k)$$

每个类别都有自己的协方差矩阵

$$\delta_k(x) = \log \pi_k - \frac{1}{2}\log |\Sigma_k| - \frac{1}{2}(x - \mu_k)^\top \Sigma_k^{-1} (x - \mu_k)$$

展开二次型：

$$\delta_k(x) = -\frac{1}{2}x^\top \Sigma_k^{-1} x + \mu_k^\top \Sigma_k^{-1} x - \frac{1}{2}\mu_k^\top \Sigma_k^{-1} \mu_k - \frac{1}{2}\log|\Sigma_k| + \log \pi_k$$

注意这里的 **$-\frac{1}{2}x^\top \Sigma_k^{-1} x$ 依赖于类别 $k$**，不能抵消 于是比较两个类别时，边界方程中会保留二次项：

$$x^\top A x + b^\top x + c = 0$$

所以 QDA 的边界是**二次超曲面** 在二维中可能是椭圆、双曲线或抛物线

QDA 相较于 LDA 更加灵活
QDA 每个类别有不同协方差矩阵，因此表达能力更强：但代价是参数更多 每个 $\Sigma_k$ 是 $d \times d$ 对称矩阵，参数数目大约是：$\frac{d(d+1)}{2}$

如果有 $K$ 个类，就要估计 $K$ 份
当维度高、样本少时，估计会很不稳定

LDA 则更受限
LDA 强迫所有类共享同一协方差矩阵，参数少, 因此偏差更大，但方差更小,估计更稳定：

所以：

- 数据足够多、各类形状确实差异明显时，QDA 可能更好
- 数据较少、共享协方差近似合理时，LDA 往往更稳

这和 Naive Bayes 逻辑相通：**模型越灵活，越可能降低偏差，但也越可能提高方差**

PS: LDA 有一个常见混淆：分类里的 LDA vs 降维里的 LDA
- **含义 1：Linear Discriminant Analysis**
    这里讲的是分类模型，是生成式分类器
- **含义 2：Fisher Linear Discriminant**
    这里常指一种监督降维/投影方法，通过最大化类间散度与类内散度比值来找投影方向
### 参数估计分析
**均值估计**

$$\hat{\mu}_k = \frac{1}{n_k} \sum_{i:y^{(i)}=k} x^{(i)}$$

类别 $k$ 的样本均值

**QDA 的协方差估计**
对每个类别单独估计：

$$\hat{\Sigma}_k = \frac{1}{n_k} \sum_{i:y^{(i)}=k} (x^{(i)} - \hat{\mu}_k)(x^{(i)} - \hat{\mu}_k)^\top$$

**LDA 的共享协方差估计**
LDA 认为所有类共享同一协方差，因此把各类样本的类内散布合并：
$$\hat{\Sigma} = \frac{1}{n} \sum_{k=1}^K \sum_{i:y^{(i)}=k} (x^{(i)} - \hat{\mu}_k)(x^{(i)} - \hat{\mu}_k)^\top$$
