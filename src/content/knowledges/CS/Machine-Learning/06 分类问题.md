## 分类的任务本质
回归里，我们关心的是：$x \mapsto y,\quad y \in \mathbb R$
给定输入 $x$ ，预测一个连续数值
分类里，我们关心的是：$x \mapsto y,\quad y \in \{1,\dots,K\}$
给定输入 xxx，从有限个类别中选一个
输出从实数换成了离散标签，但本质变化更深：
> **分类不是单纯预测“值”，而是在做“决策”**

分类任务的目的是学习一个规则$$h(x):\mathcal X\rightarrow\mathcal Y$$把输入空间划分成若干块决策区域

二分类问题的形式:
学习一个判别函数 $f(x)$
然后输出类别
$$
h(x)= \begin{cases} +1,& f(x)\ge 0\\ -1,& f(x)<0 \end{cases}
$$
这里：
- $f(x$) 叫 score / margin / logit / decision function，视模型而定
- $h(x)$ 才是最终分类结果
+ **score和label**不是同一个东西

决策边界:表示两类之间“刚好难分”
Eg.线性分类器：$f(x)=w^\top x+b$
那么决策边界就是：$w^\top x+b=0$
**分类不是在预测一个离散数字，而是在学习一个把输入映射为行动选择的决策规则**

**Q:为什么不能直接用Regression硬做Classification?**
 二分类标签用 $0/1$ 编码, 然后直接做线性回归,最后设个阈值，比如 0.5，大于 0.5 判 1，小于 0.5 判 0
 + 标签编码是任意的,但是regression会把标签当成数值真相
 + 分类只关心边界 但是回归关心离label有多远,对正类样本 y=1,f(x)=0.51和10 平方损失会严重惩罚10,因为离1很远,但实际反而这个更加确定
- 回归：关心点和目标值之间的距离
- 分类：关心点是否在边界正确一侧，以及离边界有多稳
## 分类问题的优化目标
分类真正想要优化的目标  **误分类率**,也就是 0-1 损失
+ 定义:0-1损失 分对了就是0 分错了就是1
$$
\ell_{0-1}(y,h(x))= \begin{cases} 0,& h(x)=y\\ 1,& h(x)\neq y \end{cases}
$$
也可以写成：$$
\ell_{0-1}(y,h(x)) = \mathbf 1\{h(x)\neq y\}
$$
那么experience risk就是:$$
\sum_{i=1}^n \mathbf 1\{h(x_i)\neq y_i\}
$$也就是训练集上的误分类个数, 如果除以 n，就是误分类率

对于二分类的0-1损失:
我们也可以这样写$\mathcal Y=\{-1,+1\}$，也可以写成：$$

\ell_{0\text{-}1}(h(x),y)=\mathbf 1[yh(x)\le 0]

$$$yf(x)$ 大于0代表同号 表示预测正确 异号表示预测错误
若 $y f(x)$ 很大且为正，不仅正确，而且“离边界更远”
如果很接近0 说明离边界很近,决策不稳定

但是有一个核心问题 **这个损失是很难进行优化的**
+ $\ell_{0-1}(t)=\mathbf 1[t\le 0]$ 这是一个阶跃函数 t=0时不连续的 而且大部分区域梯度是0 跳变点又没有常规梯度
+ 他只关心对错,你区分你错的多不多 对于-0.01和-100他一视同仁 没有连续反馈
+ 目标函数通常是非凸、组合性的,例如考虑一个线性分类器 经验 0-1 风险变成：$$
\hat R_{0\text{-}1}(w,b)=\frac{1}{n}\sum_{i=1}^n \mathbf 1[y_i(w^\top x_i+b)\le 0]]$$非凸非连续有大量平坦区域 本质上是一个离散计数问题 本质上是一个困难的非凸优化问题
**0-1 损失适合作为最终评价标准，但通常不适合作为直接训练目标**
最符合任务语义 但不是最适合直接训练
## surrogate loss 
0-1 损失只关心 $yf(x)$ 是否跨过 0 太硬了 我们需要一个连续的惩罚
记$z=yf(x)$ 我们把分类的loss记作$\phi(z)$ 其中$\phi$ 是一个**单调递减函数**
**surrogate loss** 的统一形式通常写成：$$\ell_{\phi}(f(x),y)=\phi(yf(x))
$$其中$\phi:\mathbb R\to\mathbb R_+$ 是某个替代函数
- $z$ 很小甚至为负时，惩罚大
- $z$ 变大时，惩罚下降
- $z$ 足够大时，惩罚很小，甚至为 0
把原本的离散决策条件 $yf(x)>0$  改写成一个可连续优化的惩罚
给定训练集 $n\{(x_i,y_i)\}_{i=1}^n$，经验风险为：
$$\hat R_\phi(f)=\frac{1}{n}\sum_{i=1}^n \phi(y_i f(x_i))$$
优点:
+ 对于“错的很离谱”和“几乎分对” 给出不同的惩罚
+ 可以连续变化,可导,凸

常见的surrogate loss:
1. 感知机损失 (Perceptron Loss)$$\phi_{\text{perc}}(z) = \max(0, -z)$$
- **核心逻辑**：只要分类正确（$z > 0$），损失就是 0；如果分类错误，损失随着错误程度（距离超平面的距离）线性增加
- **特点**：简单, 缺点是它只关注“是否分对”，如果样本恰好落在边界上，它也认为损失为 0，这导致模型可能不够稳健（鲁棒性较差）
2. Hinge Loss (合页损失)
$$
\phi_{\text{hinge}}(z) = \max(0, 1 - z)
$$
- **核心逻辑**：它是感知机损失的“加强版”。它不仅要求分对（$z > 0$），还要求分得“够稳”，分类间隔必须大于 1。
- **含义**：
    - 如果 $z \ge 1$，损失为 0
    - 如果 $0 < z < 1$，虽然分类正确，但离边界太近，依然会有少量损失
- **应用**：这是 **支持向量机 (SVM)** 的核心损失函数，旨在寻找“最大间隔”
3. Logistic Loss (对数损失)

$$\phi_{\text{log}}(z) = \log(1 + e^{-z})$$

- **核心逻辑**：它是一个平滑的、严格递减的曲线
- **特点**：
    - **处处可导**
    - **概率解释**：它将模型的输出映射到了概率空间。即使 $z$ 很大（分类非常正确），它依然会给出一个极小的损失，促使模型尽可能压低误差
- **应用**：它是 **逻辑回归 (Logistic Regression)** 的基础，也是交叉熵损失在二分类下的表现形式
4. Exponential Loss (指数损失)$$\phi_{\text{exp}}(z) = e^{-z}$$
- **核心逻辑**：当 $z$ 减小时（分类错误或间隔变小），损失呈指数级增长
- **特点**：
    - **惩罚极强**：它对离群点（Outliers）或分类错误的样本非常敏感，会迫使模型去关注那些难以分类的样本。
    - **连续可微**：数学性质良好
- **应用**：这是 **AdaBoost** 算法中默认使用的损失函数

## 线性分类器与决策边界
+ 实值打分函数 $f(x)=w^Tx+b$
+ 分类决策 $h(x)=\text{sign}(w^Tx+b)$
+ 决策边界 模型改判的地方 $f(x)=w^Tx+b=0$ 超平面
线性分类器尝试用一个单一超平面 把不同类别在feature space中区分开来
- 向量 $w$ 决定边界的方向，偏置 $b$ 决定边界的位置。
- $w^\top x+b$的符号表示样本位于边界哪一侧，$\frac{w^\top x+b}{\|w\|}$ 则对应点到边界的有符号几何距离
## 感知机
假设我们的输入特征向量为 $x \in \mathbb{R}^n$，输出类别为 $y \in \{+1, -1\}$
$$f(x) = \text{sign}(w \cdot x + b)$$
损失函数:只要分类正确，哪怕点离边界再近我也不管；我只关心那些被分错的点$y_i(w \cdot x_i + b) \le 0$
感知机的损失函数定义为**所有误分类点到超平面的总距离**
$$L(w, b) = -\sum_{x_i \in M} y_i(w \cdot x_i + b)$$

使用**随机梯度下降法 (Stochastic Gradient Descent, SGD)** 来寻找最优的 $w$ 和 $b$
## 置信度
1. 分类器的离散标签输出会丢失大量信息，因此很多模型先输出连续 score，再通过阈值或 argmax 做决策。
2. score、margin、probability 是三个不同层次：score 是内部打分，margin 是几何稳定性，probability 是统计不确定性。
3. 不是所有分类器的输出都天然是概率；感知机、SVM 更偏 score / margin，而 logistic regression、softmax regression 才有明确概率解释。
4. 概率输出的价值不只是“更细致”，而是它支持阈值调整、风险排序、拒识和代价敏感决策。
5. “置信度”这个词常常混淆内部打分强度与真实概率可靠性，必须谨慎区分。
